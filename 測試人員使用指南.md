# 🏗️ 營造業職災風險自動化評估系統 - 測試人員使用指南

## 📋 快速開始

### 第一步：下載代碼
1. 訪問: https://github.com/a0wolf013-code/-
2. 點擊綠色按鈕 **Code** → **Download ZIP**
3. 解壓到您的電腦（例如：`D:\test-system\`）

---

### 第二步：安裝依賴（只需做一次）

1. **打開 PowerShell 或 CMD**
2. 進入項目目錄：
   ```powershell
   cd D:\test-system
   ```
3. 安裝所需套件：
   ```powershell
   pip install streamlit lightgbm scikit-learn pandas numpy plotly shap
   ```

---

### 第三步：啟動系統

在 PowerShell 中運行：
```powershell
python "3_智慧前端與應用層\00_強開智慧網頁.py"
```

✅ 系統會自動：
- 啟動 Streamlit 伺服器
- 在瀏覽器中打開 http://localhost:8501
- 顯示完整的風險評估界面

---

## 🎯 測試工作流程

### ✅ 任務 1：上傳施工網圖

1. 在網頁中選擇分頁：**➕ 建立全新的工程基地項目 (NEW PROJECT)**
2. 填寫以下信息：
   - **工地名稱**：例如「測試工地 A」
   - **施工網圖**：拖放您的 Excel 進度表文件
   - **地下水位**：輸入例如 `-5.0` 公尺
   - **安全係數**：輸入例如 `1.5`
   - **廠商裁罰金額**：輸入例如 `100000` 元
3. 點擊 **🚀 確定寫入系統大腦，建立新工地資料鏈** 按鈕
4. ✅ 新工地會被保存到系統中

### ✅ 任務 2：進行風險評估

1. 切換回分頁：**🏢 選擇既有工地進行本日巡檢 (LOBBY)**
2. 從下拉選單選擇您剛才新建的工地
3. 在右邊進行配置：
   - **選擇日期**：選擇今天
   - **降雨量**：滑動設定
   - **風速**：滑動設定
   - **氣溫**：滑動設定
   - **現場缺失數**：輸入安全檢查發現的問題數
4. 系統會實時計算風險分數並顯示：
   - 🟢 綠燈（低危險）
   - 🟡 黃燈（中等危險）
   - 🔴 紅燈（高危險）

---

## 📥 上傳文件格式要求

**施工網圖 Excel 文件**
- 格式：`.xlsx` 或 `.xls`
- 內容：任何施工進度或工地資料都可以
- 系統會自動識別並提取相關信息

## ❓ 常見問題

**Q: 系統沒有啟動？**
- ✅ 確認 Python 已安裝（運行 `python --version`）
- ✅ 確認所有依賴已安裝（再次運行 `pip install` 命令）

**Q: 上傳文件後沒反應？**
- ✅ 稍等 2-3 秒讓系統處理
- ✅ 檢查 Excel 文件是否正常（不要太大）

**Q: 怎麼停止系統？**
- ✅ 在 PowerShell 中按 **Ctrl + C**
- ✅ 或直接關閉 PowerShell 窗口
