# 基於 Python 機器學習之營造業職災風險自動化評估系統開發與普適性研究

## 概述
此系統是一個完整的營造業（建築業）職災風險評估解決方案，整合了數據工程、機器學習和智慧前端，能夠自動化分析和預測施工現場的安全風險。

## 系統架構

### 第一層：數據工程與法律合規層 (`1_數據工程與法律合規層/`)
- **數據來源整合**：收集職災數據、法規條文、行政裁罰記錄等多源異質數據
- **主要模塊**：
  - `01_擷取職災數據.py` - 抓取歷史職災數據庫
  - `02_法規檔案轉換.py` - 處理法規文件並建立規則庫
  - `03_擷取裁罰紀錄.py` - 蒐集行政處罰案例
  - `04_時空模糊匹配融合.py` - 融合多源數據
  - `擷取最新職安法規.py` - 定期更新最新法規
  - `擷取歷史氣象數據.py` - 納入環境因子

### 第二層：演算法與大腦訓練層 (`2_演算法與大腦訓練層/`)
- **AI 核心引擎**：雙階段 LightGBM 機器學習模型
  - **第一階段**：重大傷亡判定模型
  - **第二階段**：嚴重度分級模型
- **特徵工程**：
  - 文本特徵挖掘與自動標籤
  - 高頻致災情境探勘
  - 嚴重度集群分析
  - 過採樣數據增強
- **風險歸因**：SHAP 特徵權重分析

### 第三層：智慧前端與應用層 (`3_智慧前端與應用層/`)
- **網頁介面** (`15_主網頁介面程式.py`) - 交互式風險評估系統
- **可視化模塊**：
  - 工地崩塌風險機率密度分佈圖
  - 工安風險 5M1E 魚骨圖分析
  - 蒙地卡羅不確定性模擬
- **報告生成**：標準 8D 缺失問題改善計畫書

## 數據資產
- **職災數據庫**：營造業 2016-2026 年歷史職災案例
- **法規庫**：全國施工安全法規大數據庫
- **行政裁罰紀錄**：歷史違規處罰案例
- **融合母表**：多源異質大數據融合結果

## 技術棧
- **後端**：Python 3.x
- **機器學習**：LightGBM、Scikit-learn、SHAP
- **數據處理**：Pandas、NumPy
- **前端**：HTML、Flask/Django（推測）
- **數據存儲**：CSV、JSON、SQLite

## 使用方式

### 安裝依賴
```bash
pip install -r requirements.txt  # 需要建立 requirements.txt
```

### 執行流程
1. 運行數據工程層的 Python 腳本以構建數據庫
2. 運行演算法層進行模型訓練
3. 啟動前端應用進行風險評估
